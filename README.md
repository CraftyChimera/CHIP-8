# CHIP-8 Interpreter

A CHIP-8 interpreter written in C++

## Prerequisites

SFML and cmake are required for this project.Install them using

```bash
sudo apt-get install libsfml-dev make
```

## Running

Place your rom in tests/ folder. Then You can run the rom using

```bash
make all
./main (rom)
```
